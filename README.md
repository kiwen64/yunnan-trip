# 云游手记

一个不需要构建工具的响应式云南旅行 PWA。桌面端同时展示地图与路线，手机端通过底部导航切换地图、行程和须知。

## 本地预览

不要直接双击 `index.html`，请在此目录启动静态服务器：

```powershell
npx serve .
```

打开终端显示的本地地址即可。地图、天气和高德导航需要联网；浏览器定位在手机上需要 HTTPS。

## 发布到手机访问

将此目录发布到任意静态托管服务（GitHub Pages、Cloudflare Pages、Netlify 等）。发布后用手机浏览器打开 HTTPS 地址，可添加到主屏幕。

如果只想临时在手机上看，可让手机与电脑连接同一 Wi-Fi，然后在本目录运行：

```powershell
python -m http.server 4173 --bind 0.0.0.0
```

用 `ipconfig` 查看电脑的 IPv4 地址，手机访问 `http://电脑IPv4地址:4173`。这种方式要求电脑保持开机，而且普通 HTTP 可能无法使用手机定位；正式旅行仍推荐发布为 HTTPS。

## 编辑与跨设备

- 页面中的“编辑行程”“编辑当天”和地点详情里的“编辑信息”会保存到当前浏览器。
- “分享行程”会生成包含当前行程数据的链接，可发到另一台设备继续查看和编辑。
- 各设备会把修改保存在自己的浏览器中；分享链接传递的是当时的行程快照，并非实时双向同步。
- 也可以通过“导出 JSON / 导入 JSON”备份或迁移。

示例数据在 `data.js`，首次打开或点击“恢复示例”时使用。
